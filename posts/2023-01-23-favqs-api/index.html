FavQs API

23 Jan 2023 web dev API

FavQs allows you to collect, discover, and share your favorite quotes. Creating an account is free, and using their API is also free. The [terms of use](API Terms of Service) are reasonable and okay.

Using the API #

QOTD #

The API can used without an account or a token for the Quote of the day:

curl https://favqs.com/api/qotd
